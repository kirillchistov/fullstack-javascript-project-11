## RSS агрегатор (MVC)
- RSS Reader — сервис для агрегации RSS-потоков, с помощью которых удобно читать разнообразные источники, например блоги. Он позволяет добавлять неограниченное количество RSS-лент, сам их обновляет и добавляет новые записи в общий поток.
- В этом проекте MVC реализуется на основе материалов [курса по архитектуре фронтенд-приложений](https://ru.hexlet.io/courses/js-frontend-architecture) и [автоматного программирования](https://ru.hexlet.io/link/D3Pwxm). Код разделяется на слои, выделяется и нормализуется состояние.
